# AI Cover Letter Generator

AI Cover Letter Generator is a simple AI-powered application that helps users quickly create professional cover letters for job applications.
The app uses artificial intelligence to generate personalized cover letters based on the user's job role, skills, and experience.

## Features

* Generate professional cover letters instantly
* User-friendly interface built with Streamlit
* AI-powered text generation
* Fast and easy to use

## Tech Stack

* Python
* Streamlit
* AI API (for text generation)

## How to Run

1. Clone the repository
2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run main.py
```
